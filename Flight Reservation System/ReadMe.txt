IMPORTANT: to be able to use SQLite in eclipse you need to set the build path,
otherwise project (Main class) will not run.

To import SQLite jar file in your Eclipse IDE, follow the steps given below.

1.Right click on your project
2.Select Build Path
3.Click on Configure Build Path
4.Click on Libraries and select Add External JARs
5.Select the jar file from the required folder
6.Click and Apply and Ok
